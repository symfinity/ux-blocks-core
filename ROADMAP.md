## Details

### 0.1.x — First public release

Atomic Twig components, role CSS, handbook and Flex recipe

### 0.2.x — CSS and typography

Native CSS interactions, flash stack, icon slots, typography roles

### 0.3.x — Component completeness

Expanded atomic component catalog, missing HTML primitives, additional variants, consistent component contracts, registry coverage improvements

### 0.4.x — Accessibility

ARIA refinements, keyboard navigation improvements, focus management, screen reader validation, WCAG compliance review

### 0.5.x — Composition

Shared composition patterns, layout interoperability, slot refinements, consistent content APIs, cross-package integration

### 0.6.x — Theming

Theme hooks, design token alignment, CSS variable refinements, appearance consistency, theme customization improvements

### 0.7.x — Customization

Override mechanisms, extension points, component configuration, rendering flexibility, project-level customization

### 0.8.x — Documentation

Component handbook, usage examples, best practices, migration guides, showcase improvements, reference documentation

### 0.9.x — Stabilization

API review, deprecation cleanup, behavioral consistency, test coverage expansion, performance refinements, release preparation

### 1.0.x — Stable API

Semver-stable component contracts, stable customization API, long-term compatibility guarantees, complete handbook
