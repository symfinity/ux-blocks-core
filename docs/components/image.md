# image

**Role:** `image`  
**Fragment id:** `blocks.image`  
**Twig:** `Image`

Responsive content images with Bootstrap-aligned presentation variants.

| Variant | Use |
|---------|-----|
| `fluid` (default) | Responsive — scales with parent width |
| `thumbnail` | Bordered thumbnail padding |
| `rounded` | Rounded corners |

Compose inside `AspectRatio` when a fixed ratio crop is required. Use `Avatar` slot for profile photos.

REF: [Bootstrap Images](https://getbootstrap.com/docs/5.3/content/images/) (semantics only).
